# 腕上课程表

> 建议使用 [AstroBox](https://astrobox.online/downloads) 安装

## 特点
- 以卡片形式显示每日课程及详细信息
- 可自由切换显示上/下一天课程
- 当课程正在进行时显示进度条
- 点击课程卡片进入全屏显示

> [!tip]
> 粗略实现多屏适配，当前只在小米手环 9 上测试无问题

## 开发
- 安装依赖
  ```
  npm install
  ```
- 调试
  ```
  npm run start
  ```
- 构建
  ```
  npm run build
  ```
- 发布
  ```
  npm run release
  ```
