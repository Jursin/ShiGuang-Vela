# 拾光课程表手环版

> 仅测试了小米手环 9
>
> 建议使用 [AstroBox](https://astrobox.online/downloads) 安装

## 开发
- 安装依赖
  ```
  npm install
  ```
- 调试
  ```
  npm run start
  ```
- 构建
  ```
  npm run build
  ```
- 发布
  ```
  npm run release
  ```
