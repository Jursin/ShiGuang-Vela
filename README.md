  <img src="get-it-on-bandbbs.png" height="80px">
</a>

## 特点
- 以卡片形式显示每日课程及详细信息
- 每日课程分时间段显示
- 可自由切换显示上/下一天课程
- 点击课程卡片进入全屏显示
- 显示上/下课倒计时和进度条

> [!tip]
> 在模拟器上测试过小米手环 8Pro/9/9Pro/10 无显示问题，但只在小米手环 9 上实机测试过

## 开发
- 安装依赖
  ```
  npm install
  ```
- 调试
  ```
  npm run start
  ```
- 构建
  ```
  npm run build
  ```
- 发布
  ```
  npm run release
  ```
